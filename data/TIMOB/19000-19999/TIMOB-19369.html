Description

1. Create a default classic & alloy app. 2. Add this to the tiapp.xml so that we know for sure we are building for API 23:

    <manifest>
        <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="23"/>
    </manifest>
</android>
